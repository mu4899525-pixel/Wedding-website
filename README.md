# 💍 3D Wedding Invitation Website

A modern, elegant, and fully responsive **3D Wedding Invitation Website** built using HTML, CSS, and JavaScript.  
This project creates a premium digital wedding experience with beautiful animations, interactive sections, and a romantic UI design.

## ✨ Features

- 💌 Animated 3D Wedding Invitation Cover
- 🎨 Premium glassmorphism and modern UI design
- 🌸 Floating hearts and particle animations
- ⏳ Live wedding countdown timer
- 📅 Wedding event details section
- 💑 Bride & Groom introduction section
- 📖 Love story timeline animation
- 🖼️ Responsive wedding gallery
- 🎵 Background music integration
- 📱 Fully responsive design for mobile, tablet, and desktop
- ✨ Smooth scrolling and transition effects
- 🎭 Interactive buttons and hover animations
- 📍 Location and event information section
- 📝 RSVP invitation section
- 🌙 Modern theme styling

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- CSS Animations
- JavaScript DOM Manipulation
- Responsive Web Design

## ⚙️ How The Algorithm Works

The website works through a combination of HTML structure, CSS styling, and JavaScript logic.

1. HTML creates the complete structure of the wedding website.
2. CSS controls the visual design, animations, layouts, and responsiveness.
3. JavaScript handles interactive features such as:
   - Countdown timer calculation
   - Dynamic animations
   - User interactions
   - Button actions
   - Page transitions

The animation system uses timed events and DOM manipulation to create a smooth and interactive user experience.
